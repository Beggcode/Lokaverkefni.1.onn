A responsive React/TypeScript application for exploring recipes using _TheMealDB API_.

## Features

- **Browse & Search:** Browse or search a wide variety of recipes.

- **Filter:** Sort recipes by _Category_ or _Country_ using a navigation bar.

- **Detailed View:** Full recipe details including ingredients, instructions, and embedded YouTube tutorials.

- **Look and Feel:** Responsive design, "Similar Recipes" suggestions, and a custom 404 error page where Gordon Ramsay berates you.

## Setup in Terminal

1. `npm install`
2. `npm run dev`
3. Press `o + enter` to open in browser
